Regular Select Github

Regular Select (JSON)

title
The Label of the Input
name
The name of the Input
id
The id of the Input and the "for" Label Attribute

To call the component

<!-- Select -->
<c3m-select
  title="Choose something"
  name="Choose Something"
  [id]="'choice-something'">
</c3m-select>

Datalist Github

Datalist - She integrates the auto-completion system. (JSON)

title
The Label of the Input
name
The name of the Input
list
Link between input and datalist
id
The id of the Input and the "for" Label Attribute

To call the component

<!-- Datalist Select -->
<c3m-select-datalist
  title="Choose Me"
  name="Choose Me"
  list="mylist1"
  [id]="'choice-me'">
</c3m-select-datalist>

Combined Datalists Github

Combined Datalists - You can associate two input to the same datalist. Both datalists integrate the auto-completion system (JSON)

title1
Label of the first Input
title2
Label of the Second Input
name1
Name of the first Input
name2
Name of the first Input
title1
Label of the first Input
id1
ID of the first Input and "for" label attribute
id2
ID of the Second Input and "for" label attribute
list
Link between the DataList ID and the "list" attributes of the two lists

To call the component

<!-- Datalists Select -->
<c3m-select-datalists
  title1="Choose Me Please"
  title2="Choose Me To"
  name1="Choose Me1 "
  name2="Choose Me2"
  list="mylist2"
  [id1]="'choice-me1'"
  [id2]="'choice-me2'">
</c3m-select-datalists>
