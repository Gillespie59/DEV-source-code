Regular Checkbox Github

Regular Checkbox. Need to call in a form tag

The Custom Class CSS version (add the custom class)

title
The label tag
name
The name of input
id
The most important attribute : Link between Label and Input
class
optional : Only for a custom css style

To call the component

<!-- SIMPLE CHECKBOX -->
<c3m-checkbox
  title="Do you want check me ?"
  name="checkboxName1"
  [id]="'checkbox1'">
</c3m-checkbox>


<!-- CUSTOM CSS CHECKBOX -->
<c3m-checkbox
  title="Do you want check me ?"
  name="checkboxName2"
  [id]="'checkbox2'"
  class="custom">
</c3m-checkbox>
