How to install Angular 2 ?

  • First Step

    The first Step is to download Node.js to use npm

    Npm is the package manager forJavaScript and the worldâ€™s largest software registry

  • Install NPM

    If you don't have the last version of Npm, you can download it with this command :

    npm install npm@latest -g

  • Install Angular

    The second Step is to download Angular-Cli with npm

    Angular-Cli is a command line interface for Angular. It allows to use

    ng new ng generate ng serve

  • Update

    To get the last Andular-Cli version

    npm install -g angular-cli@latest

How to create your project ?

  • New Project

    To create a new project, write this command :

    ng new angular-app-name

  • New Project with SASS

    If you want an Angular project that use SASS, write this command :

    ng new angular-app-name --style = scss

  • New Component

    To create a component, use :

    ng generate component my-component-name

    You can use ng generate or just ng g

  • The others ng g attributes are :

    ng g component ng g directive ng g pipe ng g service ng g class ng g guard ng g interface ng g enum ng g module

How to start your project ?

  • Git Import

    If you get your project with Github, the folder doesn't contains an node-modules folder.

    To download it, write this command :

    npm install

  • Start Project

    To test your project, use :

    ng serve

How to import CreamNG into your project ?

  • Installation

    Install via npm :

    npm install creamng

  • Import Me !

    Then Include the CreamNG module in your app root module

    import { CreamNgModule } from 'creamng';
