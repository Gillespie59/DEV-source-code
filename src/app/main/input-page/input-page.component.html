Regular Input Github

Regular Input (dont miss to add in a form tag)

title
The label tag
placeholder
Placeholder value (optional)

To call the component

<!-- SIMPLE INPUT -->
<c3m-input
  title="Entrez votre nom"
  placeholder="placeholder"
  name="name"
  id="name">
</c3m-input>

Input Floating Label Github

Input with a Floating Label bottom

Input with a Floating Label right

title
The label tag
placeholder
Placeholder value 'bottom', 'right' or 'top'

To call the component

<!-- FLOATING BOTTOM INPUT -->
<c3m-input-floating
  title="Entrez votre nom"
  floating="bottom"
  name="name1"
  id="name1">
</c3m-input-floating>

<!-- FLOATING RIGHT INPUT -->
<c3m-input-floating
  title="Entrez votre nom"
  floating="right"
  name="name1"
  id="name1">
</c3m-input-floating>
