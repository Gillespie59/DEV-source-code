Regular Input Github

Regular Input (dont miss to add in a form tag)

title
The label tag
placeholder
Placeholder value (optional)

To call the component

<!-- SIMPLE INPUT -->
<c3m-input
  title="Enter your name"
  placeholder="placeholder"
  name="name"
  [id]="'name'">
</c3m-input>

Input Floating Label Github

Input with a Floating Label bottom

Input with a Floating Label right

title
The label tag
placeholder
Placeholder value 'bottom', 'right' or 'top'

To call the component

<!-- FLOATING BOTTOM INPUT -->
<c3m-input-floating
  title="Enter your name"
  floating="bottom"
  name="name1"
  id="name1">
</c3m-input-floating>

<!-- FLOATING RIGHT INPUT -->
<c3m-input-floating
  title="Enter your name"
  floating="right"
  name="name1"
  id="name1">
</c3m-input-floating>

Input Error Message Github

Input Error Message (dont miss to add in a form tag) under an Input Tag / Component

The field is not correct
Between the tags
Insert the text error

To call the component

  <div class="containerForDemo">
  <!-- SIMPLE INPUT -->
  <c3m-input
    title="Enter a name"
    placeholder="just look error"
    name="nameBeforeError"
    [id]="'nameBeforeError'">
  </c3m-input>

  <!-- INPUT ERROR MESSAGE -->
  <c3m-input-error-msg>
    The field is not correct
  </c3m-input-error-msg>
</div>
