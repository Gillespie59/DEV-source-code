Display the current page in the app hierarchy. Each Item is a page. The last Item is the current page.

To call the component 

    <!-- BREADCRUMBS -->
    <c3m-breadcrumbs></c3m-breadcrumbs>

In your App Module 

  import { BreadcrumbsModule } from './module-breadcrumbs';

  imports: [
      RouterModule.forRoot(appRoutes),
      BreadcrumbsModule
  ]

The Breadcrumbs module 

  export * from '../app/main/breadcrumbs-page/breadcrumbs/breadcrumbs.component' ;
  export * from '../app/main/breadcrumbs-page/breadcrumbs/breadcrumbs.service' ;
  export * from '../app/main/breadcrumbs-page/breadcrumbs/breadcrumbs.model' ;
  export * from '../app/main/breadcrumbs-page/breadcrumbs/breadcrumbs.module' ;
