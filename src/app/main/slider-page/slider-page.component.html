Regular Slider Github

Regular slider with OUTPUT tag.

title
The label tag
min
Minimum slider value
max
Maximum slider value
step
Steps value (Integer or Float)
value
Default value

To call the component

  <!-- SIMPLE SLIDER -->
    <c3m-slider
          title="Simple Slider"
          min="0"
          max="50"
          value="45"
          step="1">
    </c3m-slider>

Tooltip Slider Github

The current value is displayed in a sliding tooltip.

Just add the class="bubble" when calling the c3m-slider component to display the OUTPUT element as a tooltip.
