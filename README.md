Installation
=====================

Run
`npm install creamngnpm` 

In your app.module.ts
`import { CreamNG } from 'creamngnpm';`
And
`@NgModule({
  declarations: [
    CreamNG
   ]
  })`

HELP US
=====================

To create a component
`ng generate component component name`
